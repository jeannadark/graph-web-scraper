Guidelines

  1. Login/Register
  2. Click Domains button(there are three of them but only domains one is necessary which is blue colored)
  3. Fill the fields
    1. About - briefly indicate what the domain is about
    2. Category - select 1 or 2 options from the list. Help text below will help you to select more than 1 item.
    3. Url - for this one click plus sign and insert the url and then click save. If domain already exist, or the url scheme is not correct, or domain does not exist, there will be error message indicating the issue.

      ***Domain existence validation can take some time please be patient, if it takes too much close the pop up window and try again.

    4. Subdomains - for this one click plus sign and insert the url and then click save. See help text below for the syntax.
    5. Languages - select 1 or 2 options from the list. Help text below will help you to select more than 1 item.
  4. Click save and in case of successful submission, a success message will be popped out otherwise, an error message indicating the problem.
